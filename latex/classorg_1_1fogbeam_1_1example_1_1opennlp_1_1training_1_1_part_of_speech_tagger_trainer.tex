\doxysubsection{Detailed Description}
Entrena un modelo de etiquetado gramatical (POS) con datos de entrenamiento. 

Esta clase utiliza Open\+NLP para entrenar un modelo de etiquetado gramatical (Part-\/of-\/\+Speech Tagging) basado en datos de entrenamiento en formato especificado. Una vez entrenado, el modelo se guarda para su uso futuro.
